Σελίδα του API του Vaseis-app

Πανεπιστήμια

GET /universities -> επιστρέφει το σύνολο των πανεπιστημίων

GET /universities/{uni-id} -> επιστρέφει το πανεπιστήμιο που αντιστοιχεί στο id

Βάσεις

GET /bases/{year} -> επιστρέφει τις βάσεις της χρονιάς που δόθηκε

GET /bases/dept/{dept_id} -> επιστρέφει τις βάσεις του τμήματος που αντιστοιχεί στο id

GET /bases/{year}/dept/{dept_id} -> επιστρέφει τις βάσεις του τμήματος που αντιστοιχεί στο id για τη χρονιά που δόθηκε

Τμήματα

GET /departments -> επιστρέφει το σύνολο των τμημάτων όλων των πανεπιστημίων

GET /departments/{dept_id} -> επιστρέφει το τμήμα που αντιστοιχεί στο id

GET /departments/university/{uni_id} -> επιστρέφει το σύνολο των τμημάτων που ανήκουν στο πανεπιστήμιο στο οποίο αντιστοιχεί το id

Τύποι Εξέτασης

GET /examtypes -> επιστρέφει το σύνολο των τύπων εξέτασης

Ειδικές Κατηγορίες

GET /special-categories -> επιστρέφει το σύνολο των ειδικών κατηγοριών

Στατιστικά

Δεν έχει υλοποιηθεί ακόμα
